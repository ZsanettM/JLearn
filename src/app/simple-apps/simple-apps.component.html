Very Simple Programs

Python makes it easy to run single lines of code-one-liner programs. Let's give it a go.

Start Here! Code Example 1 
{{"
            >>> print ('Hello, world!'')
            "}}

What happened? You just created a program, that prints the words 'Hello, world!'.

Note that in Python 2 the parentheses are optional after print, but in Python 3.0 they are required.

Try to insert a new cell into the notebook on the right and enter the example code! Then press run: It will immediately compile whatever you have typed in. This is useful for testing things, e.g., defining a few variables, and then testing to see if a certain line will work.
Don't type the >>> part!
